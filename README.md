# manifold calibration

This repository is used to sync this folder across different environments.

## Repository

Remote repository:

```bash
https://github.com/MarcusBen/manifold-calibration
```

Currently synced scope:

- Entire project folder
- Temporary and editor-generated files are still ignored by `.gitignore`

## Daily sync

Pull the latest changes:

```bash
git pull
```

Commit and push your local changes:

```bash
git add .
git commit -m "update"
git push
```

## New environment

Clone the repository:

```bash
git clone https://github.com/MarcusBen/manifold-calibration.git
```
