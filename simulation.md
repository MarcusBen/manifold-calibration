## 一、建议的总实验流程

### 1. 数据准备层

用完整 HFSS 角扫数据作为“真值流形库”：

- 全角域响应：$\Theta_{\text{all}}$
- 稀疏校准子集：$\Theta_c$
- 未见测试集：$\Theta_{\text{test}}=\Theta_{\text{all}}\setminus\Theta_c$

这里最关键的是：**训练角和测试角必须严格分离**。这本来就是你计划书里的核心卖点之一。

### 2. 流形建模层

统一做三种流形：

- 理想流形 $\tilde a_I(\theta)$
- HFSS 真值流形 $\tilde a_H(\theta)$
- 你方法得到的修正流形 $\hat a(\theta)$

最好再加一个简单基线：

- 残差直接插值法（比如线性插值 / spline）

这样你才能证明“你的低维物理建模”不只是“随便插值一下”。

### 3. 数据生成层

DOA 仿真时，**接收快拍必须始终由 HFSS 真值流形生成**，即：
$$
X = A_H(\theta)S + N
$$
然后分别用：

- 理想流形做估计
- 你的修正流形做估计
- HFSS 真值流形做估计（oracle 上界）

这一点很重要。否则如果生成和估计都用同一个修正流形，会有“自证正确”的嫌疑。

### 4. 评价层

分成两级：

**一级：流形级指标**

- steering vector 相对误差
- 复相关系数
- 相位残差 RMSE
- 幅度残差 RMSE
- 未见方向平均误差 / 最大误差

**二级：DOA 级指标**

- RMSE vs SNR
- RMSE vs snapshots
- 成功率（如误差 < 1° / 2°）
- 分辨概率
- 伪谱峰值位置、主瓣宽度、副瓣高度

## 二、必须做的核心 case

下面这些 case，我建议你按“必做核心 + 可选增强”来安排。

------

### Case 1：问题成立性验证

**目的**：证明“理想流形失配”这个问题是真的存在，不是为了做算法硬造问题。

**实验环境**

- 用完整 HFSS 流形和理想流形逐角比较
- 角域用你当前 HFSS 可稳定扫描的范围，例如 $[-60^\circ,60^\circ]$ 或你现在已有的有效范围
- 对每个角度比较 8 个阵元的幅度/相位差
- 再做单源 MUSIC，对比 ideal-manifold estimator 和 HFSS-oracle estimator

**输出图**

- 每个阵元的相位残差随角度变化图
- 幅度残差随角度变化图
- steering vector 相似度曲线
- 单源伪谱对比图
- 高 SNR 下的角度偏差图

**你要得到的结论**

- 理想流形与 HFSS 流形之间存在系统性偏差
- 用理想流形做 MUSIC 会出现谱峰偏移/误差地板
- 单纯提高 SNR 不能消掉这种模型失配误差

这个 case 对应你计划书里的“问题成立性”和“高 SNR 误差地板”论点。

------

### Case 2：失配主导成分分析

**目的**：证明你采用“相位主导修正”不是拍脑袋。

**实验环境**

- 计算
  $$
  r(\theta)=a_H(\theta)\odot a_I^*(\theta)
  $$
  并分离幅度和相位残差

- 比较三种近似：

  1. 只修正相位
  2. 只修正幅度
  3. 幅相都修正

**输出图**

- 相位残差能量占比
- 幅度残差能量占比
- 三种模型在未见方向上的流形误差柱状图
- 三种模型的 DOA RMSE 对比图

**你要得到的结论**

- 当前一维阵列场景下，相位残差是主导项
- 相位修正已经能恢复大部分性能
- 幅度联合修正可以作为后续增强，而不是第一版必须项

这正好对应算法里“phase-dominant correction”的核心假设。

------

### Case 3：未见方向流形泛化验证

**目的**：这是你整篇文章最核心的 case。

**实验环境**

- 从全角域中均匀抽取少量校准角，比如
  - $L=5$
  - $L=9$
  - $L=13$
- 其余全部角度作为未见测试角
- 拟合 $\phi_m(u)$，其中 $u=\sin\theta$
- 比较 ideal / interpolation / proposed / HFSS

**输出图**

- 未见角度上的平均流形误差曲线
- 每个阵元的相位重构曲线（真实 vs 拟合）
- 若干代表性角度的 steering vector 对比

**你要得到的结论**

- 少量校准样本下，你的方法可以在未见方向上明显优于 ideal
- 与简单插值相比，你的方法更稳、更平滑、更符合物理结构
- 与 HFSS oracle 的差距是可控的

如果这个 case 做不好，后面所有 RMSE 图都站不稳。

------

### Case 4：校准样本数量敏感性

**目的**：证明“少量”到底少到什么程度还有效，这是你的工程价值所在。

**实验环境**
 固定：

- 拟合阶数 $P$
- 正则参数 $\lambda$
- 均匀分布采样

变化：

- $L = 3,5,7,9,13,17$

**输出图**

- 未见方向流形误差 vs $L$
- 单源 DOA RMSE vs $L$
- 双源分辨概率 vs $L$

**你要得到的结论**

- 存在一个“性能拐点”
- 从极少样本到中等样本，性能快速提升
- 超过某个 $L$ 后收益趋于饱和

这能直接支撑“低校准负担”这个卖点。

------

### Case 5：校准样本分布敏感性

**目的**：证明不仅样本数量重要，样本布局也重要。

**实验环境**
 固定 $L$，例如 $L=9$，设计 4 种校准策略：

1. 均匀覆盖
2. 中央密集
3. 边缘增强
4. 随机采样

**输出图**

- 各策略下的未见方向流形误差
- 各策略下 RMSE vs SNR
- 箱线图或均值±标准差图

**你要得到的结论**

- 均匀覆盖通常最稳
- 边缘区失配若更严重，可适当边缘加密
- 中央扎堆会损害全角域泛化

这个 case 很重要，因为它会让你的工作更像“校准设计方法”，而不只是“拟合模型”。

------

### Case 6：模型阶数与正则参数敏感性

**目的**：证明你这个低维模型不是靠凑阶数凑出来的。

**实验环境**

- 基函数：多项式 / Chebyshev
- 阶数：$P=1,2,3,4,5$
- 正则：$\lambda$ 取若干档

**输出图**

- 未见方向误差热图：$(P,\lambda)$
- 过拟合现象示意图
- 两类基函数对比图

**你要得到的结论**

- 低阶就能达到较好效果
- 阶数过高会过拟合校准角、损害未见角
- 正则化能稳定外推

这和你算法里的正则最小二乘闭式解是直接对应的。

------

### Case 7：单源 DOA，RMSE vs SNR

**目的**：这是最标准的性能图，但不要把它当唯一主图。

**实验环境**

- 单源
- 源角度只选**未见测试角**
- SNR: $-15:5:20$ dB 或你现在常用范围
- snapshots: 固定一个中等值，比如 200 或 500
- Monte Carlo: 200~500 次

比较对象：

- Ideal manifold + MUSIC
- Proposed corrected manifold + MUSIC
- HFSS oracle manifold + MUSIC
- 可选：插值基线

**输出图**

- RMSE vs SNR
- 成功率 vs SNR
- 若干代表 SNR 下的伪谱图

**你要得到的结论**

- 低 SNR 时噪声主导，几种方法差距有限
- 中高 SNR 时模型失配主导，ideal 出现误差地板
- proposed 明显逼近 HFSS oracle

这会是你论文里最直观的一张主图。

------

### Case 8：单源 DOA，RMSE vs snapshots

**目的**：证明模型修正和统计快拍效应是不同层面的事。

**实验环境**

- 固定 SNR，例如 0 dB 和 10 dB 两档
- snapshots: $N=50,100,200,500,1000$
- 源角为未见角

**输出图**

- RMSE vs snapshots
- 高快拍区误差地板对比图

**你要得到的结论**

- 快拍数增加能压低统计误差
- 但 ideal 的模型误差不会因快拍无限增多而消失
- proposed 可显著降低高快拍下的偏差地板

这个 case 很能说明“你解决的是模型误差，不是噪声误差”。

------

### Case 9：双源分辨力测试

**目的**：证明修正流形不仅平均误差更低，而且伪谱更“像样”。

**实验环境**

- 双非相干源
- 对称布置，例如 $\pm 5^\circ$、$\pm 8^\circ$、$\pm 10^\circ$
- 或固定中心角，改变间隔 $\Delta=2^\circ,4^\circ,6^\circ,8^\circ$
- SNR 固定为 0 或 5 dB
- snapshots 固定为 500

**输出图**

- 分辨概率 vs 角间隔
- 典型伪谱图
- 峰值偏差统计

**你要得到的结论**

- ideal 流形下主瓣会更宽、峰更偏
- proposed 下双峰更容易分开
- 说明你的方法提升的不只是“平均 RMSE”，还提升了谱质量

你计划书里明确提到了“MUSIC 伪谱质量”应作为考察对象，这个 case 正好落实。

------

### Case 10：随机划分稳健性

**目的**：避免别人质疑“你是挑了一组好看的校准角”。

**实验环境**

- 固定 $L$
- 随机抽取校准集 50 次或 100 次
- 每次重做训练和测试
- 统计均值和标准差

**输出图**

- 箱线图
- 均值±标准差误差条

**你要得到的结论**

- 方法对校准集选择不是极端敏感
- 但样本分布确实影响性能
- 这进一步支持 Case 5 的结论
